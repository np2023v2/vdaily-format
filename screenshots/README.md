## Adding Screenshots

For WordPress.org submission, add screenshots named:
- `screenshot-1.png` - Main demo view
- `screenshot-2.png` - Code block examples
- `screenshot-3.png` - Admin interface (if applicable)
- `screenshot-4.png` - Mobile responsive view

Screenshots should be:
- PNG or JPG format
- At least 1200px wide
- Aspect ratio of approximately 4:3 or 16:9
